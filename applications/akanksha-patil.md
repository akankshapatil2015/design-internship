## 4. A project you've shipped recently

https://antigravity-clone-psi.vercel.app/

This is an interactive physics simulation inspired by Google’s Antigravity experiment. I built it using vanilla JavaScript and Matter.js, implementing realistic gravity, drag interactions, and dynamic rendering using the Canvas API. The project focuses on creating an interactive and intuitive user experience with features like gravity toggle, explosion, and drag-and-throw interactions. If I revisit it, I would improve performance for larger element counts and enhance mobile touch interactions.
